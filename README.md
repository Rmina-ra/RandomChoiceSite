# RandomChoiceSite

Простое приложение на Java/Spring Boot, которое показывает случайную картинку при нажатии на кнопку.

##  Технологии
- Java 17
- Spring Boot
- Thymeleaf
- HTML/CSS

##  Как запустить

1. Клонируй репозиторий
2. Открой в IntelliJ IDEA
3. Запусти `RandomChoiceSiteApplication.java`
4. Открой браузер: http://localhost:8080
